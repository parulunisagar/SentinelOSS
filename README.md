# SentinelOSS
Open-source Linux server monitoring and security analytics platform built in C++17. Monitor system health, detect security events, analyze logs, and manage infrastructure from a unified dashboard.

Sentinel OSS is an open-source security and infrastructure monitoring platform designed for Linux servers. It provides real-time system monitoring, security event tracking, log analysis, alerting, and a web dashboard to help developers, students, and system administrators better understand and secure their environments.
